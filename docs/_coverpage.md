# **Dyna Boot**

> 基于 **dyna-framework** 的官方示例项目

- 最新技术栈抢先体验
- 通用增删改查导能力支持
- 内置动态鉴权与方法级权限控制
- 遵循企业级开发规范，能够快速落地业务

## 技术栈

- **基础框架**：Spring Boot4 · JDK21
- **安全认证**：Spring Security · JWT
- **数据访问**：Mybatis-Plus · Redis&nbsp;&nbsp;&nbsp;
