## 代码生成

> 基于 Mybatis-Plus 定制的代码生成器。

- 自动生成 CRUDE 接口、服务类、数据访问接口、mapper.xml、Entity、DTO、VO、Convertor、HttpClient、方法权限等。

```pom
<dependency>
    <groupId>com.dynaframework</groupId>
    <artifactId>dyna-generator</artifactId>
    <scope>provided</scope>
</dependency>
```

## 配置项

全部可配置项见 GeneratorProperties 配置类。
